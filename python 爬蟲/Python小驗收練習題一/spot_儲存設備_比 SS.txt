"隨著影像圖片解析度與檔案大小不斷成長，儲存裝置（尤其是可攜式儲存裝置）能否趕上腳步迎合成長需求很重要。SD 協會（SD Association）19 日發表最新 SD 8.0 規格，SD Express 卡在 PCIe 4.0 強力支援下，傳輸速將超越固態硬碟（SSD）達到驚人的 4GB/s。SD 8.0 標準仍採用 NVMe Express 協定，但使用雙通道 PCIe 4.0 連接獲得更多頻寬，這也是何以能達到 4GB/s 的原因。更精準說，全新 SD Express 卡是透過記憶體卡背面利用單或雙 PCIe 4.0 通道的全新針腳布局，達到如此高的傳輸速度。此速度是目前 SD Express 卡速度（985MBps）4 倍左右，更別說最快的 UHS-III 高速 SD 記憶卡（624MBps）更難望其項背，甚至 PC 常見的 SSD 硬碟也被遠遠抛在後面。（Source：SD 協會）值得一提的是，4GB/s 最高速度必須搭配使用支援雙道通 PCIe 4.0 的讀卡機才能辦到，如果使用 PCIe 3.0，傳輸速度將減半至約 2Gbps。即使如此，仍比最快的 SD 卡快 2 倍。當然，SD 8.0 記憶卡也回溯相容較舊規格的讀卡機，只不過速度會大打折扣。此外，SD 協會表示，全新 SD Express 將適用不同記憶卡容量規格，包括 SDHC、SDXC 和 SDUC，所以理論上未來可買到最高 128TB 的 SD 8.0 記憶卡。隨著 4K / 8K 電視、手機與相機興起，這些裝置會以更高的解析度與更快的傳輸速率拍攝照片與影片，所以 SD 8.0 記憶卡便顯得十分重要。尤其對有超高畫質拍照或攝影，以及有大量 RAW 格式照片連拍需求的專業攝影師來說，SD 8.0 記憶卡能提供極大幫助。此規格對任天堂 Switch 之類的遊樂器同樣助益良多，因為這些裝置多半會透過個別 MicroSD 插槽擴充原本僅 32GB 的內建儲存容量，透過 SD 8.0 不但可提升容量，更能提升整體遊戲載入與執行速度。雖然 SD 協會正式宣布 SD Spec 8.0，但短時間市面還不會有支援新標準的記憶卡（最快也要到 2020 年底才有可能看到），畢竟三星（Samsung）、Sony 等 SD 卡製造商需要一些時間，才能在新 SD Express 卡支援 SD 8.0 標準。（圖片來源：SD 協會）"
